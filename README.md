# 💰 BudgetBuddy

> Smart expense & budget management for students aged 16+

BudgetBuddy helps students track daily expenses, plan monthly budgets, monitor savings goals, and build better financial habits — all in one modern dashboard.

---

## 🚀 Tech Stack

| Layer      | Technology               |
|------------|--------------------------|
| Frontend   | React.js, Bootstrap 5, Chart.js, Axios |
| Backend    | Python Flask             |
| Database   | MySQL                    |
| Tools      | VS Code, XAMPP, MySQL Workbench, Git |

---

## 📄 Pages

### Before Login
- **Home Page** (`/`) — Introduction, Features, About, Contact, Login & Register buttons

### Auth
- **Login** (`/login`) — Email + password sign in
- **Register** (`/register`) — Create account (age 16+ required)

### After Login *(coming soon)*
- Dashboard, Add Expense, Expense History, Reports & Charts, Savings Goals, Notifications, Profile

## 🛠️ Getting Started

### Frontend

```bash
cd frontend
npm install
npm start
```

### Backend *(Flask — coming soon)*

```bash
cd backend
pip install -r requirements.txt
python app.py
```
